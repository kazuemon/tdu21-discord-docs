# キャンパスマップ

Discord キャンパスでは各チャンネルを4ケタでナンバリングしています。
例) 1101 ... 1号館 1階 1号室

TIP

追加してほしいチャンネルがあれば #4101_事務部受付 よりお気軽にご連絡ください！

注意

#学科設定 にて学科を設定後、順次チャンネルが表示されます。
学科設定後も表示されない場合は、モデレーターの個人チャットまでお問い合わせください。
